# IKÇÜ – Nanobilim ve Nanoteknoloji Lisansüstü Programları (Statik Site Taslağı)

Bu klasör saf HTML/CSS/JS içerir. Bir CMS'ye (ASP.NET / custom CMS) entegre edilebilir veya
doğrudan yerelde çalıştırılabilir.

## Yerelde çalıştırma
- `index.html` dosyasını çift tıklayıp açabilirsiniz.
- Eğer bazı tarayıcılarda gömülü içerikler (iframe) kısıtlanırsa basit bir yerel sunucu kullanın:

  Python:
  `python -m http.server 8000`

  Sonra:
  `http://localhost:8000`

## İçerik durumu
- Menü ve sayfa iskeletleri, “NANO Prog Web Sitesi Planı” belgesindeki site haritasına göre hazırlandı.
- “Program Çıktıları” ve “Dersler/Müfredat” sayfaları ekli metinlerden dolduruldu.
- Akademik kadro sayfasında sadece CV dosyasında bulunan kısıtlı metinler kullanıldı; e-posta/ABD/ders listeleri yer tutucu.

## Güncellenmesi gerekenler
- Resmî misyon/vizyon, program tanıtım metni (onaylı versiyon)
- Akademik kadro iletişim bilgileri + anahtar kelimeler + seçili yayınlar
- Laboratuvar listesi, cihaz listesi ve görseller
- Tez listesi (Excel/Word'den import edilebilir)
- Kurumsal logo ve resmî renk kodları (CSS değişkenleri ile kolayca güncellenir)
